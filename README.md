# 🌌 Thapar Atlas — 100% Vibe-Coded

Welcome to **Thapar Atlas**, the premium, all-in-one student portal for **Thapar Institute (TIET)**. 

This entire platform was built with **zero manual coding**. Every styling token, Firestore query, security rule, and AI moderation layer was **vibe-coded** using conversational agentic AI assistants.

---

## 🚀 Key Features

*   **📚 Resource Hub**: Access and share academic study materials, notes, and assignments.
*   **📝 PYQ Hub**: Search and download Previous Year Question papers indexed by branch, semester, and course.
*   **🗺️ Interactive Campus Map**: High-fidelity rotatable vector map of the TIET campus with searchable academic blocks and facilities.
*   **💻 Admin Analytics Dashboard**: Real-time monitoring of active viewers via an automated student session heartbeat tracker.
*   **🛡️ Production-Grade Security**: Strict Content Security Policy (CSP), rate-limiting throttles, and row-level Firestore rules to block script injection and IDOR.
*   **🔒 Automated AI Moderation**: Automated screening of uploaded academic content utilizing generative language intelligence.
*   **📶 WiFi & Classroom Trackers**: Report WiFi speeds and locate available classroom slots instantly.

---

## 🛠️ Tech Stack

*   **Frontend**: React (Vite), CSS Custom Properties, Lucide Icons
*   **Backend & DB**: Firebase Authentication & Cloud Firestore
*   **Uploads Hosting**: Cloudinary
*   **Configuration & Routing**: `vercel.json` rewrite engine
